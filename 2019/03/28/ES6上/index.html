Symbol

Symbol 是 ES6 中新增的一种原始类型。

Symbol 类型通过调用 Symbol() 函数生成，每一个 Symbol 类型的值都是唯一的。

let、const

var 的特点

  • 支持变量声明预解析
  • 不支持块级作用域
  • 可以重复声明

let、const 的特点

  • 不支持变量声明预解析 (先声明后使用)
  • 支持块级作用域
  • 不能重复声明 (暂时性死区)

不支持变量预解析和不支持变量的重复声明大大地提高了数据使用的安全性。所以 var 能不用则不用。

const 常量

  • 声明时就必须初始化
  • 初识化后就不能改变

const 在存对象的时候，存的是对象的引用(地址)，引用不能改变，但对象的内容却可以。
